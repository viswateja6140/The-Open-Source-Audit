## Software Chosen
Linux Kernel

## Scripts

### Script 1
Displays system information like kernel version, user, uptime, and OS details.

### Script 2
Checks if the kernel package is installed and displays its details.

### Script 3
Audits important directories for size and permissions.

### Script 4
Analyzes log files and counts occurrences of errors.

### Script 5
Generates a personalized open-source manifesto.
